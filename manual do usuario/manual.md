## PARA O USUÁRIO
- Para se cadastrar, o usuário deve clicar no botão Cadastrar, no canto superior direito da página inicial, que o redirecionará para a página de cadastro. 
Chegando na página de Cadastro, haverá campos que pedirão informações do usuário como nome, cpf, endereço, entre outros. Preencha os campos corretamente (caso algo dê errado, haverá um aviso sobre o que está errado, leia e corrija).

 ![img1](https://github.com/cp2-dc-info-projeto-final/pizzaria-favattos-portugal/blob/master/manual%20do%20usuario/img1.jpeg)
 
 - Ao clicar em cadastrar-se, você será redirecionado para a página de Login, onde você vai precisar informar seu nome de usuário (ou email) e sua senha (os preenchidos no cadastro).
 
 ![img2](https://github.com/cp2-dc-info-projeto-final/pizzaria-favattos-portugal/blob/master/manual%20do%20usuario/img2.jpg)
 
 - Dali, você será redirecionado novamente à página inicial do site, onde estarão os produtos a serem comprados. 
Para comprá-los, basta clicar na opção de tamanho do produto (haverá o preço explicitado), o produto vai ser direcionado ao carrinho, onde você poderá observar tudo o que você quer comprar até agora, podendo, inclusive, alterar, remover ou, claro, adicionar.

![img3](https://github.com/cp2-dc-info-projeto-final/pizzaria-favattos-portugal/blob/master/manual%20do%20usuario/img3.jpg)

- Para comprar, finalize o pagamento presente no carrinho e será direcionado à uma página onde precisará preencher algumas informações como forma de pagamento, e caso queira, também poderá adicionar observações para o seu pedido. Basta finalizar e estará pronto.

![img4](https://github.com/cp2-dc-info-projeto-final/pizzaria-favattos-portugal/blob/master/manual%20do%20usuario/img4.jpg)

- O usuário tem uma página própria, onde contém suas informações e uma página para caso queira alterá-las, basta clicar em "Editar Perfil" e substituir as informações antigas por atuais.

![img5](https://github.com/cp2-dc-info-projeto-final/pizzaria-favattos-portugal/blob/master/manual%20do%20usuario/img5.jpg)

- Haverá, no site, uma página específica para fotos postadas pelo proprietário, basta clicar em "Fotos" na parte superior do site e será direcionado a um site com diversas fotos.

![img6](https://github.com/cp2-dc-info-projeto-final/pizzaria-favattos-portugal/blob/master/manual%20do%20usuario/img6.jpg)

## PARA O ADMIN
- O Admin deverá se cadastrar como qualquer outro usuário, e receberá permissão de admin através de um outro admin que poderá lhe conceder a permissão. 
- Com a permissão de administrador, ele poderá alterar dados dos produtos, adicionar novos produtos, alterar produtos, preços e descrições na página inicial.

![img7](https://github.com/cp2-dc-info-projeto-final/pizzaria-favattos-portugal/blob/master/manual%20do%20usuario/img7.jpg)

![img8](https://github.com/cp2-dc-info-projeto-final/pizzaria-favattos-portugal/blob/master/manual%20do%20usuario/img8.jpg)

- Além disso, o admin terá um “Histórico de pedidos”, para chegar até a página, clique em Usuário, no canto superior direito, que abrirá algumas opções, dentre elas, ‘Histórico de pedidos”. Ali, ele poderá observar os pedidos que estão sendo feitos, com as informações necessárias para realizar as vendas, como cliente, endereço, produtos e preço somado.

![img9](https://github.com/cp2-dc-info-projeto-final/pizzaria-favattos-portugal/blob/master/manual%20do%20usuario/img9.jpg)
